# Eva System - Web API (.NET 8)

Eva System is a multilingual Web API built with **ASP.NET Core (.NET 8)** for managing ingredients linked to specific regions. These ingredients are then used to create products. The system also supports **dynamic language translation** using **Google Translate**.
